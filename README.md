# BeastFightingChess

#### 斗兽棋
##### 分为红蓝两方，每一方分别拥有 象，狮，虎，豹，犬，狼，狗，猫，鼠 八张牌，在一个 4 X 4 的棋盘中覆盖。

##### 象，狮，虎，豹，犬，狼，狗，猫，鼠 从大到小，大的可以吃掉小的，但是鼠鼠可以吃掉大象，

##### 一方的棋子被吃光了，游戏就结束，对方胜利
